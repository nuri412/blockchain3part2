## - Usage

### 1. Install Dependencies
```powershell
npm install
```
### 2. Compile Contracts
```powershell
npx hardhat compile 
```
Or just save Solidity contracts (Ctrl + S)
### 3. Run Tests
```powershell
npx hardhat test
```
